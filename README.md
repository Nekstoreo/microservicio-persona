# Microservicio Persona

Gestión de personas del sistema.

## Requisitos

- Java 25+
- MySQL 8.0+
- Gradle 9.3.0+

## Levantar el servicio

```bash
./gradlew bootRun
```

El servicio estará disponible en `http://localhost:8081`

**OpenAPI (Swagger):** `http://localhost:8081/swagger-ui.html`
